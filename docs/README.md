## Project Stack

-   **Runtime**: Bun
-   **Framework**: Next.js 15 with App Router
-   **Database**: InstantDB (real-time graph database)
-   **Authentication**: Better Auth
-   **Styling**: Tailwind CSS v4
-   **Icons**: Phosphor Icons
-   **Deployment**: Ubuntu VPS with Nginx
